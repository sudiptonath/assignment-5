What is the Document Object Model (DOM) ?

The Document Object Model (DOM) defines webpage elements and their structure.

How do you select an element from DOM ?

Use document.querySelector or document.getElementById,etc to select elements.

What is event delegation in the context on the DOM , and why is it useful ?

Event delegation is a technique where a single parent element handles events for its child elements.

How do you manipulation an elements attributes and styles using the DOM.

Manipulate attributes with `element.setAttribute`, and modify styles using `element.style.property` in JavaScript.
